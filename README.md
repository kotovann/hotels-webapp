!Для запуска Docker используйте команды Makefile!

####  Первый запуск (с загрузкой фикстур):

```bash
make init
```

#### Запуск с пересборкой + миграции (без загрузки фикстур):

```bash
make dev
```

#### Просто запуск сущесвуещих контейнеров:
```bash
make up
```

#### Пересоздать БД:

```bash
make db-redo
```

#### Остановить:

```bash
make down
```

Остановить и удалить данные БД:

```bash
make down-v
```
